## 基础使用指南

### 1. 查看 VPS 信息
登录后依次点击：
1. "Services" → "My Services"
2. 选择对应 VPS 进入 KiwiVM 控制面板

### 2. 重要参数获取
- **SSH 端口**：非默认端口，需在面板查看
- **IP 地址**：控制面板首页显示
- **root 密码**：可通过邮件或面板重置

### 3. 系统重装步骤
1. 点击"Stop"停止服务器
2. 选择"Install new OS"
3. 从列表中选择所需系统（推荐 CentOS/Debian）
4. 确认后自动开始安装
